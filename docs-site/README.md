# Gess docs site

An [Astro Starlight](https://starlight.astro.build) site that publishes the
Markdown guides in `../docs/` as a searchable, navigable web site.

## Canonical source

`../docs/*.md` is canonical. Do not hand-edit files under
`src/content/docs/` — that directory is generated by
`scripts/sync-docs.mjs` and is ignored by Git. To add, rename, or reorder a
page, edit the `pages` list in `scripts/sync-docs.mjs` and the matching
Markdown file in `../docs/`.

The sync step runs automatically before `dev` and `build`. To run it by
hand:

```sh
npm run sync
```

## Commands

Run from this directory:

| Command           | Action                                       |
| :----------------- | :------------------------------------------- |
| `npm install`       | Install dependencies                          |
| `npm run sync`      | Regenerate `src/content/docs/` from `../docs/` |
| `npm run dev`       | Start the local development server at `localhost:4321` |
| `npm run build`     | Sync, then build the static site to `./dist/`  |
| `npm run preview`   | Preview a production build locally             |
